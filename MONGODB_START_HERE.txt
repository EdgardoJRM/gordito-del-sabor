⚡ INICIO RÁPIDO (7 PASOS - 17 MINUTOS)

1. Crea cuenta en MongoDB Atlas
   👉 https://www.mongodb.com/cloud/atlas

2. Crea un cluster M0 (gratis)

3. Crea usuario: gordito_user

4. Permite acceso desde cualquier IP

5. Obtén tu connection string

6. Agrega a .env.local:
   MONGODB_URI=mongodb+srv://gordito_user:password@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

7. Prueba:
   npm run dev

🆘 PROBLEMAS?

Error: "MONGODB_URI is not defined"
  → Verifica que .env.local tenga MONGODB_URI

Error: "Connection refused"
  → Verifica que tu IP esté permitida en Network Access

Error: "Authentication failed"
  → Verifica username, password y que el usuario esté creado

Error: "Timeout"
  → Verifica tu conexión a internet y que el cluster esté activo
